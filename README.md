# PERSONAL PORTFOLIO SITE

## Development
  ```sh
  yarn dev 
  ```
## Formatting
  ```sh
  yarn format
  ```
## Build
  ```sh
  yarn build
  ```
## Styling
SCSS files can be found in the `src/assets/scss/` directory. The folders are structured as follows:
* `/base`
  * Global properties
* `/components`
  * Component specific styles
* `/mixins`
  * SCSS mixins
